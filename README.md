# Machine-Learning-Linear-Regression-Project
# 📊 E-Commerce Customer Analysis using Linear Regression
Linear Regression model to analyze customer behavior and predict yearly spending, helping an e-commerce company decide between app vs website optimization.

## 📌 Problem Statement
An e-commerce company wants to decide whether to focus more on improving their **mobile app** or **website** to increase customer spending.

---
## 📂 Dataset
The dataset contains customer behavior and spending information:

- Avg. Session Length  
- Time on App  
- Time on Website  
- Length of Membership  
- Yearly Amount Spent (Target Variable)

---

## ⚙️ Approach

1. Data Loading & Cleaning  
2. Exploratory Data Analysis (EDA)  
3. Correlation Analysis (Heatmap)  
4. Feature Selection  
5. Model Building using Linear Regression  
6. Model Evaluation (MAE, MSE, R²)  
7. Residual Analysis  

---

## 📊 Key Insights

- Strong correlation between **Length of Membership** and spending  
- Moderate impact of **Time on App**  
- Very low impact of **Time on Website**  

---

## 📈 Model Performance

- Model evaluated using:
  - Mean Absolute Error (MAE)
  - Mean Squared Error (MSE)
  - R² Score

- Residuals showed near normal distribution → good model fit

---

## 💡 Business Conclusion

- Customers who stay longer (membership) spend more  
- Mobile app contributes more to revenue than website  

👉 **Recommendation:** Focus on improving the mobile app experience
